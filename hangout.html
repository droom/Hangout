hangOut.js

A jQuery plugin for hanging quotations on the tumblr platform

I've seen a thousand tumblr themes. I've not seen one with hanging punctuation. This is because tumblr's generated markup makes it difficult to do. This plugin provides a CSS hook by applying a .hangOut class to relevant quote marks. You can then style them as you wish.
